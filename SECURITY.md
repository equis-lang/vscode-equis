# Security Policy

## Supported Versions
Security updates are provided for the latest stable release.

| Version | Supported |
| ------- | --------- |
| v0.1.0  | YES       |

## Reporting a Vulnerability

If you discover a potential security issue in the compiler or runtime—especially regarding duality checks or memory safety—please do not open a public issue.

Instead, please send a detailed report to **lintangmaulanazulfan@gmail.com**. Include a minimal example that demonstrates the issue and allow us time to investigate and provide a fix before any public disclosure.
